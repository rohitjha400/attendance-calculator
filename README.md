A simple web application that helps students calculate their attendance percentage and check exam eligibility.

## Problem Statement
Students often struggle to quickly calculate their attendance percentage to know whether they meet the minimum attendance requirement.

## Features
- Input total number of classes
- Input number of classes attended
- Calculates attendance percentage
- Shows eligibility based on 75% criteria

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in any web browser
